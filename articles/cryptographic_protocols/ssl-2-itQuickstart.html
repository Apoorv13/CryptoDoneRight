WebServers:

In order to disable SSLv2 on WebServers:
If you are a server administrator, you really should disable SSLv2 for the security of your users. By doing this, clients will no longer be able to use the insecure SSLv2 protocol to connect to you, and will have to use a newer, more secure/recommended alternative.

Disable SSLv2 in Windows: http://support.microsoft.com/kb/187498
Disable SSLv2 in Apache: http://adamyoung.net/Disable-SSLv2-System-Wide


Browsers:

All of the popular modern browsers do not support SSLv2 anymore.
